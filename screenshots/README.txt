# Screenshots Folder

This folder is intended for storing app screenshots for the GitHub repository.

Please add your app screenshots with the following names:
- app_logo.png - The app's logo/icon
- screenshot1.png, screenshot2.png, screenshot3.png - Main gallery screenshots
- chats_screen.png - Screenshot of the Chats tab
- chat_detail.png - Screenshot of an individual conversation
- status_screen.png - Screenshot of the Status tab
- calls_screen.png - Screenshot of the Calls tab

Once added, these screenshots will be displayed in the README.md file.
